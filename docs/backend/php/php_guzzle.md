## 简介

Guzzle 是一个 PHP HTTP 客户端，支持发送 HTTP 请求。它提供了一个简单的接口来构建和发送 HTTP 请求，并处理响应。Guzzle 支持多种 HTTP 方法（如 GET、POST、PUT、DELETE 等），并且可以轻松地处理请求参数、头信息和响应数据。
