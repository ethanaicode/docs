# 日语

## 五十音

五十音是日语的音节表，由五十个音节组成。五十音表中的每个音节都是一个音节，即一个元音或一个辅音加一个元音。五十音表中的每个音节都有一个对应的假名，即一个汉字或假名符号。

### 五十音字源

日文字都是以汉字为基础的，所创造出的五十音假名（借字）最初目的只是为了标示发音，而不是为了书写。

平假名是由中文的草书演变而来的，而片假名则是由中文的楷书演变而来的。

### 五十音输入法

#### 如何打出 `かっこいい`

在日语输入法（九键或全键盘）中，打出 **かっこいい** 的关键在于那个缩小的 **「っ」**。

这个小「っ」在日语中叫做**促音**。用普通电脑键盘（QWERTY全键盘）打出这个词，主要有两种非常简单的方法：

- 双写下一个音的辅音字母（最常用、最快），这是最地道、最顺手的打法。只要**把小「っ」后面那个字母的第一个拼音字母输入两次**就可以了。连起来打就是：`k akko ii`

- 单独输入小字，可以在这个音的罗马字前面加上 **`l`** 或 **`x`**（代表 Little 或 eXtra）。

  连起来打就是：`k a` + `l t u` + `k o` + `i` + `i`
